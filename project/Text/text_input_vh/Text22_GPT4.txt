The load switching device has its own communication interface and connects with the control panel.
The relays in the load switching device follow a program specified by the central system.
The central system must override the circuit program with spontaneous commands.
The next command changes the state of the switching device.
Each relay's position must be reported back to the central system.
The relay settings must be displayed on site.
The central system sends a command to override a relay.
Meanwhile, the service interface overrides a relay.
The relay in the load switching device switches to the desired status and reports back to the central system.
The relay's position on the load switching device is visible and logged in a logbook.
